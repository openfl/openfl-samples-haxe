OpenFL Samples (Haxe)
=====================

## Get Started

### Clone the repository

```bash
git clone https://github.com/openfl/openfl-samples-haxe
```

### Choose a sample

```bash
cd openfl-samples-haxe/features/display/DisplayingABitmap
```

### Install and start

```bash
npm install
npm start
```
